# 🚀 Incog

> Anonymous Career Intelligence & Growth Platform for Students

Incog is a full-stack web platform designed to help students grow professionally without the fear of judgment.  
It combines anonymous discussions, structured roadmaps, company-wise interview intelligence, and powerful career tools into one unified ecosystem.

---

## 🌟 Vision

Incog empowers students to:
- Ask doubts anonymously
- Prepare for placements strategically
- Access structured learning roadmaps
- Analyze company-wise interview trends
- Use career-building tools like resume builder & OA simulator

---

## 🧠 Core Features

### 🔐 Anonymous Community
- Anonymous profiles
- Post, comment, and react
- Skill-based topic tagging
- Company-based discussions

### 📚 Structured Roadmaps
- Web Development
- Data Structures & Algorithms
- Core CS Subjects (OS, DBMS, CN, OOPS)
- Internship Preparation
- Placement Preparation

### 📊 Company Intelligence System
- Hiring pattern breakdown
- Topic distribution analysis
- Frequently asked interview questions
- Difficulty analysis
- Salary insights

### 🛠 Career Tools Suite
- ATS-Friendly Resume Builder
- Salary Trajectory Analyzer
- Online Assessment (OA) Simulator
- Interview Readiness Analyzer
- Progress Dashboard

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository


git clone https://github.com/techyfatcat/Incog.git

cd Incog


---

### 2️⃣ Setup Backend


cd server
npm install


Create `.env` file inside `server/`:


PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


Start backend:


npm start


---

### 3️⃣ Setup Frontend


cd client
npm install
npm run dev
